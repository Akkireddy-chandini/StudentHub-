# StudentHub – Student Management System

A full-featured Student Management System developed using HTML, CSS, JavaScript, PHP, MySQL, and XAMPP.

## Features

* Student Dashboard
* Faculty Dashboard
* Attendance Management
* Task Tracking System
* Timetable Management
* Notes Management
* Secure Authentication System
* Database Integration

## Technologies Used

* HTML5
* CSS3
* JavaScript
* PHP
* MySQL
* XAMPP

## Project Type

Individual MCA Project

## Modules

### Student Module

* View timetable
* Check tasks
* Access notes
* Track attendance

### Faculty Module

* Manage student records
* Upload notes
* Assign tasks
* Manage attendance

## Installation Steps

1. Install XAMPP
2. Move project folder to `htdocs`
3. Import SQL database in phpMyAdmin
4. Start Apache and MySQL
5. Open project in browser using localhost
